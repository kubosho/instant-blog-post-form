# Instant blog post form

Blog post form for blog.kubosho.com.

![Page screenshot](https://cdn.jsdelivr.net/gh/kubosho/instant-blog-post-form@master/screenshot/v1.0.0.png?version=1.0.0)

## Development

Launch development server:

```
npm run dev
```

Execute build:

```
npm run build
```

Run test runner:

```
npm test
```
